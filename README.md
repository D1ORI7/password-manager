Password Manager Lite
A simple console-based password manager written in C++.
It allows you to store website credentials, generate random passwords, search entries, delete them, and save everything to a file for later use.
Features
Add a new entry (site, login, generated password)
Generate a random password of custom length
View all saved entries
Search entries by site name
Delete entries by site
Load data from file on startup
Save all data to file on exit
Usage
Run the program.
Choose an option from the menu:
1 Add entry
2 Show all entries
3 Find by site
4 Delete by site
5 Save and exit
All changes are saved to passwords.txt.
Technologies
C++
STL (vector, string, fstream)
Basic file I/O
Random password generation
